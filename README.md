# AI Time Server

Простой HTTP-сервер, который отдаёт текущее время как **plain text** — без JavaScript.

Это позволяет языковым моделям, умеющим делать HTTP-запросы, узнать актуальное время,
даже если они не могут выполнять JS в своей среде.

## Эндпоинты

| URL | Формат | Описание |
|-----|--------|----------|
| `/` | `text/plain` | Время в читаемом виде (UTC) |
| `/json` | `application/json` | Машиночитаемый JSON |

### Пример ответа `/`
```
Текущее время (UTC): понедельник, 11 мая 2026 г., 14:32:07
ISO 8601:            2026-05-11T14:32:07.123Z
Unix timestamp:      1747053127
```

### Пример ответа `/json`
```json
{
  "utc": "2026-05-11T14:32:07.123Z",
  "unix": 1747053127,
  "timezone": "UTC"
}
```

## Деплой на Render

1. Загрузи код в GitHub-репозиторий
2. Зайди на [render.com](https://render.com) → **New → Web Service**
3. Подключи репозиторий
4. Настройки:
   - **Runtime:** Node
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** Free
5. Нажми **Deploy** — готово!

Render сам подставит переменную `PORT`, сервер её подхватит автоматически.
